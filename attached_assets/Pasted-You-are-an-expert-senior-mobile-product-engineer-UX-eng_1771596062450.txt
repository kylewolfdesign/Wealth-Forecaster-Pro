CORE APP CONCEPT
A net worth and wealth forecasting app. User enters assets, liabilities, and investment holdings. App shows:
- Current net worth dashboard (by category + total).
- History over time (net worth trend).
- Forecast projections (1, 5, 10, 20, 50 years) using assumed annual growth rates.
- Ability to adjust annual growth assumptions for each investment category (stocks/ETFs, crypto, RSUs, cash/savings, other assets; and mortgage rate / principal payments).

KEY REQUIREMENTS (must implement)
1) Onboarding Wizard (first launch)
- Step-by-step wizard collecting:
  A) Investments (public tickers): type ticker + number of shares.
  B) RSUs: ticker + number of shares + vesting schedule.
     - Vesting schedule model: start date, cliff (months), total duration (months), vest frequency (monthly/quarterly), and optional “already vested” shares.
  C) Savings accounts: current amount + monthly contribution (e.g., +2000/month) + optional annual interest rate.
  D) Offset accounts: current amount + monthly contribution + optional annual interest rate.
  E) Mortgage: current principal balance + interest rate + monthly payment (or principal payment growth inputs).
     - Support monthly payment amount and optional annual increase % for payment.
  F) Additional assets: name + current value + optional annual growth rate (e.g., car, collectibles, property equity, private shares).
- Wizard ends with a review screen and saves everything.

2) Post-onboarding editing
- User can add/edit/delete any category items at any time:
  - Investments holdings
  - RSUs and schedules
  - Savings accounts
  - Offset accounts
  - Mortgage
  - Additional assets
- Provide “Add” actions from each category list and from a global “+ Add” button.

3) Totals + drill-in
- A “Breakdown” screen that shows totals by category and allows drilling into category detail lists.
- Each detail list shows item rows with current value, growth assumptions, and edit action.

4) Home Dashboard
- Shows total net worth (assets - liabilities).
- Shows summary tiles by category:
  - Stocks/ETFs (holdings)
  - Crypto
  - RSUs (vested value + unvested value optionally)
  - Savings
  - Offset
  - Other assets
  - Mortgage (liability)
- Show net worth change over time using stored history (e.g., last 30/90/365 days) + a chart.
- Show “Today vs last snapshot” delta.

5) History tracking
- Maintain a daily snapshot of total net worth and category totals.
- Snapshot is recorded on app open (once per day) and whenever the user saves changes.
- Store history locally (AsyncStorage).
- Provide a History screen with a chart and list of snapshots.

6) Forecasting
- Forecast future net worth at 1, 5, 10, 20, 50 years.
- Use compounding growth assumptions:
  - Investments: annual growth % adjustable (global default + per-holding override optional).
  - Crypto: annual growth % adjustable
  - RSUs: simulate vesting into vested shares; vested shares then grow at investment growth rate for their ticker category.
  - Savings/Offset: include monthly contributions and optional annual interest rate (or use a default cash growth rate).
  - Mortgage: amortization model using balance, interest rate, monthly payment, and optional annual payment increase %. If too complex, approximate with principal reduction using payment - interest each month.
  - Additional assets: annual growth % adjustable per asset.
- Include a “Assumptions” settings screen where the user can:
  - Set default annual growth for Stocks/ETFs
  - Set default annual growth for Crypto
  - Set default annual growth for RSUs (or map to ticker type)
  - Set default annual growth for Cash (Savings/Offset)
  - Set inflation rate (optional) and a toggle to show “real” vs “nominal”
- Forecast screen should show:
  - A line chart of net worth projection over time (monthly points for first 5 years, yearly after).
  - A table/list of the 1/5/10/20/50 year values.
  - Ability to tweak assumptions inline and instantly recompute.
